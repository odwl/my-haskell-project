Usage: /google/bin/releases/gemini-cli/tools/gemini [options_for_gemini_cli]

Launches Gemini CLI for google3 development, passing along any command-line
arguments specified to this script

To see the help for Gemini CLI, run:

  /google/bin/releases/gemini-cli/tools/gemini -- --help

so that "--help" will be passed along to the application instead of being
consumed by this bash script.
