---
description: How to generate omni-rich, single-page live artifacts for the post and outline.
---

Follow these steps to generate or update the live artifacts for this project:

1. **Read Sources**: Always read the latest content from the local `docs/` directory, specifically `docs/outline.md` and `docs/post.md`.
2. **Output**:
    - Write the outline to `rich_outline_live_page.md` in the brain artifact directory as a live artifact. the artifact should contain 100% of the outline content, and be formatted in a way that is easy to read and navigate.
    - Write the post to `post_full_text_live_page.md` in the brain artifact directory as a live artifact. the artifact should contain 100% of the post content, and be formatted in a way that is easy to read and navigate.
